# Pagurai MVP Website 2026

Next.js MVP website for Pagurai Creative: AI content, automation, paid media, landing pages, and digital strategy.

## Features

- Next.js App Router.
- Fully responsive landing page.
- Dark and light themes.
- EN / ES language switcher, with English as the default language.
- Floating Cyber-Capsule navigation.
- Pagurai animated background system.
- Interactive glow cards based on the Pagurai theme references.
- Organized assets and references folders.
